Keycloak roles

Create these roles in your Keycloak realm and assign them to users or groups:

RoleAccess
vmstig-viewerView scans and checklists
vmstig-scannerViewer + create and run scans
vmstig-remediatorScanner + run remediation
vmstig-adminFull access including vCenters, schedules, this page

Keycloak client checklist

  1. Create a realm (or use an existing one) for the VMware STIG Tool.
  2. Create a confidential OpenID Connect client with standard flow enabled.
  3. Valid redirect URI: {{ callback_url }}
  4. Valid post-logout redirect: {{ post_logout_url }}
  5. Add the four vmstig-* roles above and assign them to users/groups.
  6. Ensure roles appear in the access token (realm roles or client roles for this client).
